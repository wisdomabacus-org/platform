# Wisdom Abacus Platform

This is a Turborepo monorepo powered by [Bun](https://bun.sh).

## Project Structure

- **apps/**
  - `website`: Public website application
  - `exam-portal`: Exam portal application
  - `admin`: Administration dashboard
- **packages/**: Shared packages and libraries
- **configuration/**: Project configuration files

## Getting Started

### Install Dependencies

```bash
bun install
```

### Build

```bash
bun run build
```

### Develop

```bash
bun run dev
```

### Utilities

- This repository is configured with [Turborepo](https://turbo.build/repo)
- Package manager: [Bun](https://bun.sh)
